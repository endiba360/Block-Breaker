# Block Breaker
Block Breaker is a 2D game made with Unity as a recreation of the classic "Breakout" game.
## Our Core Game Design
- **Player Experience:**
Quick reactions
- **Core Mechanics:**
Breaking blocks
- **Theme:**
Blocky, colourful
- **Core Game Loop:**
Smash all blocks without letting the ball drop, in order to progress to the next level.
